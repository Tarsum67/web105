QR Code Generator

This QR Code Generator app lets you instantly convert any text or URL into a scannable QR code with just one click. Whether you want to share a website, send someone a Wi-Fi password, or save a piece of text, this tool makes it incredibly easy. Designed for speed and simplicity, the app gives users a no-frills way to generate QR codes that can be scanned with any modern mobile device. To use the app, simply type or paste your desired content—whether it’s a URL, plain text, or contact info—into the input field. When you click the generate button, the app sends that data to a public QR code API, which returns a high-quality QR image. The image is immediately displayed on the screen and can be scanned right away or downloaded for later use. The app was built with HTML, CSS, and JavaScript, focusing on responsive design and practical functionality. Working on this project gave me valuable experience with external API integration, DOM manipulation, and UX-focused web development. It’s a perfect example of how a simple idea, when well-executed, can result in a useful and intuitive web tool.

The JavaScript handles the API call and dynamically updates the DOM to display the resulting image without needing to reload the page. The layout adjusts automatically for different screen sizes using CSS Flexbox and media queries, ensuring it’s user-friendly on both desktop and mobile devices. One of the best things about this app is how lightweight and accessible it is. There’s no login, no account needed, and no clutter—just a clean interface with a single purpose: create QR codes fast. Users can generate codes on the go, for things like restaurant menus, event RSVPs, or quick links to social media profiles.
